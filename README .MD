# Teste de Automação Inmetrics

## Instalar as dependências
```bash
bundle install
```

<br>

## Para rodar o projeto os testes
```bash
cucumber
```

<br>

## Para rodar feature especifíca
```bash
cucumber /features/<nome_da_feature>.feature
```

<br><br>

##  Screenshots das telas em falha estarão:
```
./report/screenshots
```
